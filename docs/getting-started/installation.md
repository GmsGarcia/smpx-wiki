# Installation

This page will help you get **SMPx** up and running on your Paper server.

## Requirements

Before installing SMPx, make sure you have the following requirements:

- Java 21
- Paper 1.21.10 or newer
- [Vault](https://www.spigotmc.org/resources/vault.34315/) or [VaultUnlocked](https://www.spigotmc.org/resources/vaultunlocked.117277/) (Optional, but highly recommended)

## Installing SMPx

Download the latest version from [Modrinth](https://modrinth.com/plugin/smpx-core) or [Github](https://github.com/GmsGarcia/smpx-core).

Place the .jar file into your `plugins` folder and start the server.

If you see any errors related to this plugin in the terminal, please refer to the [FAQ](../faq) page or our [Discord](https://discord.com/invite/VSgTpTGZ8A).
